Description

Determines the parameters of a function that interpolates the five-dimensional gridded data (x(i),y(j),z(k),q(l),r(m),fcn(i,j,k,l,m)) for i=1,..,nx, j=1,..,ny, k=1,..,nz, l=1,..,nq, and m=1,..,nr. The interpolating function and its derivatives may subsequently be evaluated by the function db5val.

See db3ink header for more details.
