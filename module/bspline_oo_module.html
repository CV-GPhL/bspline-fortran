Description

Object-oriented style wrappers to bspline_sub_module. This module provides classes (bspline_2d, bspline_3d, bspline_4d, bspline_5d, and bspline_6d) which can be used instead of the main subroutine interface.

History

  • Jacob Williams, created 6/20/2015.
